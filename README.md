# AI Agent Task Configurator

This is the front-end user interface for an AI Agent Task Configurator, built for the BEC Bagalkote Full Stack Development Micro Project.

## Live Demo
Check out the live demo here: [https://sangam-gaddi.github.io/agent-ui-event-handling-/](https://sangam-gaddi.github.io/agent-ui-event-handling-/)

## Features
- **Dynamic Loading Screen:** Built with GSAP for smooth animations.
- **Glassmorphism Design:** A modern dark-glassmorphism aesthetic.
- **Responsive Layout:** Adjusts gracefully across different screen sizes.

## How to Run Locally
1. Clone or download the repository.
2. If you are on a desktop, you can open `index.html` directly in your browser.
3. If you are on a mobile device, due to mobile browser security constraints, opening local files might cause errors. The local `index.html` has been configured to automatically redirect to the live GitHub pages link when opened directly from the filesystem.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- GSAP (GreenSock Animation Platform)
